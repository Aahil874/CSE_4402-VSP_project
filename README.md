# ⚾ Project Home Run! (CSE_4402 - VSP Project)

A lightweight, high-performance 2D hardware-accelerated gameplay engine engineered to deliver real-time rendering, ultra-smooth animations, and optimized frame-rate output. 

Inspired by classic retro baseball games (like *R.B.I. Baseball* and *Bases Loaded*), **Project Home Run!** modernizes old-school arcade physics using modern graphics rendering techniques.

---

## 🚀 Core Features

*   **Hardware Accelerated Engine:** Leverages GPU acceleration to ensure a steady, high-performance frame rate and smooth sprite handling.
*   **Real-Time Rendering pipeline:** Custom frame output architecture built to handle dynamic sports mechanics without screen tearing or visual lag.
*   **Retro Aesthetics:** Mimics the addictive gameplay loops and visual charm of 16-bit arcade baseball classics.
*   **Dynamic Animation Control:** Features fluid sprite sheets and physics-based interactions for pitching, batting, and running the bases.
